# 算法说明 根据密码长度进行处理
  
##当密码长度小于6时
判断需要增加字符数a；
3字符重复的次数b；
不包含数字，小写和大写的次数c;
从(a,b,c)中取大者为本次修改次数，即消耗成本S

##当密码长度在6和20之间时
3字符重复的次数b；
不包含数字，小写和大写的次数c;
从(b,c)中取大者为为本次修改次数，即消耗成本S

##当密码长度大于20时
需要删除的字符数d;
获取所有连续重复字符数>=3重复数量列表L，即aaaabbbbcccdaaaa 为[4,4,3,4]
并获取3字符重复次数b;
循环L列表除以3取余为0，则b和d都减1，并将L列表中对应的数减1，直到b或者d等于0或者列表循环结束退出循环，然后保留L中数据都大于3组成新的L；
继续循环l列表除以3取余为1，则b减1，d减2，并对l列表对应的数减2，直到b等于0或者d=1或者列表循环结束退出循环，然后保留l中数据大于3组成新的L;
然后存在多种情况：
1.b=0，d>=0,则返回需要删除的字符数D，S=D+c
2.d=0, b>0 ,则返回3字符重复的字符数B,S=B和c中的大者
3.b>0,d>0
d除以3=d1，余数为d2
if d1>=b，S=D+c
else if d2 > 0 S= (b-d1+D,c)中的大者
else S=B和c中的大者

算法处理结束
